# Sangharsh Delivery Challan

A desktop application built for Sangharsh Machinery to create, manage, and export delivery challans. Runs entirely offline. All data stays on your machine — nothing is sent to any external server.

---

## What it does

- Create and print branded delivery challans as PDFs
- Manage item inventory with pricing and stock information
- Maintain a party list (clients/recipients) for quick reuse across challans
- Dashboard view of pending and completed challans
- Backup and restore data from within the app
- Customize company name, address, logo, and challan format in Settings

## Installation

1. Download the `.exe` installer from the [Releases](https://github.com/Harshil-Anuwadia/Sangharsh-Inventory-Management-Software/releases) page or your company's distribution channel.
2. Run the installer and follow the on-screen steps.
3. Launch from the desktop shortcut or Start Menu.

Before creating your first challan, go to **Settings** and configure your company name, address, and logo.

## Data & Storage

All data is stored locally on your machine.

- **User records & settings** — saved in the `Sangharsh/` directory under your user profile
- **Exported PDFs** — go to your Documents folder by default (configurable in Settings)
- **Storage format** — lightweight JSON; easy to back up or move manually
- **External transfer** — none; no data leaves your machine

Backups are managed from **Settings → Backup & Restore**. Run a backup before any major Windows update or system migration.
